## 业务约束摘要

- CN001 免费用户每日对话上限 3 轮（hard / 已实现）
- CN002 每月连续签到恢复次数上限 1 次（hard / 已实现）
- CN003 场景发布需经审核流程（draft → review → published/rejected）（hard / 已实现）
- CN004 封禁用户操作需二次确认（confirmation body required）（hard / 已实现）
- CN005 RevenueCat Webhook 必须验证 HMAC 签名（hard / 已实现）
- CN006 用户词汇表 (user_id, word) 唯一约束（hard / 已实现）
- CN007 FSRS 间隔重复算法参数（stability/difficulty/due）自动计算（hard / 已实现）
- CN008 密码必须 bcrypt 哈希存储，不可明文（hard / 已实现）
- CN009 JWT access_token 短期有效（15-30min），refresh_token 长期有效（hard / 已实现）
- CN010 管理端操作需角色权限验证（role-based access control）（hard / 已实现）
- CN011 对话消息按 sequence 递增排序，不可乱序（hard / 已实现）
- CN012 AI 质量低分阈值可配置（默认 3.0/5.0）（soft / 已实现）
- CN013 封禁/解封操作审计日志（操作人、时间、原因）（hard / 部分实现）
- CN014 语音文件大小/时长上限限制（soft / 未实现）
- CN015 Prompt 模板版本管理和回滚机制（soft / 未实现）
